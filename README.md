<h1>Projeto de Compras APP</h1>
<p>Aplicativo desenvolvido para a aula do DEVinHouse do Senai, utilizando <strong>React-Native e JavaScript.</strong></p>

<h1>Uma visão geral do projeto</h1>
<p>Este projeto é um aplicativo de compras que permite aos usuários visualizar, adicionar e gerenciar itens em seu carrinho.</p>

<p>Esta seção fornece uma visão do design e das funcionalidades do aplicativo.</p>

<h1>Funcionalidades</h1>
<ul>
  <li>Navegar pelos itens disponíveis: Os usuários podem visualizar uma variedade de produtos disponíveis.</li>
  <li>Adicionar itens ao carrinho: Os usuários podem adicionar itens que desejam comprar.</li>
  <li>Visualizar o carrinho: Os usuários podem ver todos os itens que adicionaram ao carrinho.</li>
  <li>Interface amigável: A interface é simples e fácil de navegar.</li>
</ul>

<h1>Tecnologias Utilizadas</h1>
<ul>
  <li>React-Native: Framework para desenvolvimento de aplicativos móveis para iOS e Android.</li>
  <li>Context API: Usado para gerenciar o estado global do aplicativo.</li>
  <li>JavaScript: Linguagem de programação utilizada no projeto.</li>
  <li>Expo: Ferramenta utilizada para simplificar a emulação e os testes do aplicativo.</li>
  <li>Android Studio: Usado para emular um dispositivo Android durante o desenvolvimento.</li>
</ul>

<p>Você pode verificar todas as bibliotecas utilizadas neste projeto simplesmente conferindo as dependências no arquivo package-lock.json.</p>
<p>Certifique-se de executar <code>npm install</code>, isso instalará todas as bibliotecas utilizadas com base no que está escrito no arquivo package-lock.json.</p>
<p>Novas funcionalidades e refatorações serão adicionadas no futuro, caso sejam solicitadas.</p>
